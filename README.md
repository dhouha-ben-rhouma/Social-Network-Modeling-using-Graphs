# Social Network Graph in C

## 📌 Overview
This project models a social network using a weighted graph. Each user is represented as a vertex, and each connection is an edge with a weight indicating the strength of the relationship.  

## ⚙️ Technologies
- Language: C
- Concepts: Graphs, Data Structures, Pointers, File Handling
- Tools: Code::Blocks, GCC

## 🧠 What I Learned
- How to implement graphs in C using adjacency lists
- Working with pointers and dynamic memory allocation
- Modeling real-world data structures (social networks)
- Algorithm for traversing weighted graphs


## 🚀 How to Run
1. Clone the repository
2. Open the project in Code::Blocks (or compile with GCC)
3. Run `main.c` and follow instructions to load or add users
